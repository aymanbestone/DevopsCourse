==========================
Taint and Untaint in Terraform 

Taint ==> You can "taint" a resource using terraform taint to mark it as needing to be re=created during next terraform apply 
It's like telling Terraform that a resource is a "bad" or need re deployment.

$ terraform taint aws_instance.linux_vm

Untaint ==> Using terraform untaint, you can remove the taint from a resource so it will not be re created and will stay as it is during next apply

$ terraform untaint aws_instance.linux_vm

===========================

Terraform Modules
==================

A terraform module is a set of Terraform Configuration files in a single directory

Single Directory with one or more .tf files can be considered as a module

One Root module can have any number of child modules in terraform
example : Inside one project or module we can create multiple child modules ex2,s3,vpc........


Note : We will be running terraform commands from root module and root module will invoke child modules for execution

( Refer diagram attached in resources and Live class recordings )


==> Create project directory 

$ mkdir 05-terraform-project

==> create main.tf, provider.tf and output.tf in root directory


==> vi provider.tf

provider "aws" {
  region = "ap-south-1"
}

==> vi main.tf

module "my_ec2" {
  source = "./modules/ec2"
  ami    = "ami-0d54604676873b4ec"
}
module "my_s3" {
  source = "./modules/s3"
}


==> vi output.tf

output "ec2_vm_public_ip" {
  value = module.my_ec2.ec2_vm_public_ip
}

output "ec2_vm_private_ip" {
  value = module.my_ec2.ec2_vm_private_ip
}


==> create modules dir

$ mkdir modules

==> create s3 and ec2 child modules

$ mkdir ec2

$ mkdir s3


==> inside ec2 child module 
   

=> vi input.tf
variable "ami" {
  description = "Amazon vm image value"
  type = string
}

variable "instance_type" {
  description = "represents the type of instance"
  default     = "t2.micro"
}


=> vi main.tf

resource "aws_instance" "linux_vm" {

  ami             = var.ami
  instance_type   = var.instance_type
  key_name        = "terraform-kp"
  security_groups = ["default"]
  tags = {
    Name = "Telusko-Server"
  }
}


=> vi output.tf

output "ec2_vm_public_ip" {

        value = aws_instance.linux_vm.public_ip
}
output "ec2_vm_private_ip" {

        value = aws_instance.linux_vm.private_ip
}
