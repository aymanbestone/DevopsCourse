There are several types of IP's ( Internet protocol ) address used in a computer/machine network

IPV4
====== 
--> 32 bits 
--> 4 sets of numbers 
--> 192.168.1.1 
--> Uses Dotted decimal notation

IPV6
======
--> 128 bits 
--> 8 sets of four hexadecimal digits
--> Uses hexadecimal notation with colons 
--> 2001:0db8:85a3:0000:0000:8a2e:0370:7334

Supports 340 undecillion addresses
